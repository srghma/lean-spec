## Generated test state

The test runner writes this file in the project root:

```gitignore
.lean-spec-timings-and-last-failures
```

Add this name to the project's `.gitignore`. The name must match the filename
used by the runner; if it is changed in the code, update `.gitignore` to match
it as well.
